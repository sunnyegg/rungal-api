## Installation
### Clone
```bash
$ git clone https://github.com/sunnyegg/rungal-api.git
$ cd rungal-api
$ npm install
```

---
### Create Environment Variables
```bash
$ mv .env.example .env
$ [your text editor] .env
```

---
### Start
```bash
$ npm start
```
